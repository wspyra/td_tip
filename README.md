T+D Code challenge solution

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'td_tip'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install td_tip

## Usage

Commands:

    $ calculate-tip
    
    $ calculate-tip --amount AMOUNT --tip NUMBER

    $ calculate-tip help calculate
